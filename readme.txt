RAPPEL : Toujours refresh avant de commit /!\

Liens utiles : 
- http://www.arkhe.com/simulateur/

Pour partager vos fichiers :
1) Faire une syncronisation pour récupérer la dernière version
2) Mettre le fichier dans le dossier "JeuEntreprise"
3.A) Une fenetre nommé "uncommitted changes" apparait en haut à gauche de l'écran
3.B) Mettre un titre EXPLICITE sur la modification (exemple : "modification texte.txt")
3.C) Mettre une description plus précise si possible (ex : "Ajout de conditions sur ...")
3.D) Cliquer sur Commit, puis vous pouvez syncroniser à nouveau
